# muit-travel

三菱UFJインフォメーションテクノロジー「新規事業創出ハッカソン」

## frontend
```
docker-compose run --rm frontend npm install
```

## backend
backend/.env
```
SECRET_KEY=xxxx
DEBUG=True
```

## start containers
```
docker-compose up -d
```
* frontend
  * http://localhost:3000/
* backend
  * http://localhost:8000/
